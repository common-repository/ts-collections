TS Collections provide some usefull Wordpress Customizations, filters, actions to make your wordpress experience more smoother and user friendly. 

== Description ==
**Please drop me a message, if there is any problem in using the plugin.**

This plugin provides some common missing options in WordPress like -

1. Force All Javascript in Footer., 2. Force ASYNC for all Javascript except jQuery., 3. Force jQuery In Footer., 4. Automatically link Twitter usernames., 5. Mark Comments with Very Long URLs as Spam., 6. Set Minimum Comment Length to 20., 7. Allow PHP in Default Text Widgets., 8. Remove Auto Linking of URLs in WordPress Comments., 9. Remove the URL Field from the WordPress Comment Form.

The plugin comes with the option to enable or disable each feature.

== Installation ==

How to install?


1. Upload `the-last-one` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Or
Install it via WordPress Plugins menu.
